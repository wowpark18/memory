### 🖥️ Task 1: 사용 패턴 분석 피드백 HUD 컴포넌트 디자인 브리프

이 요소는 메인 화면(코디 제안 전/후)이나 추천 로직을 구동할 때, 사용자에게 마치 시스템 내부에서 실시간으로 정보가 분석되는 듯한 '지능형 컨설팅' 경험을 제공하는 데 사용됩니다.

#### 1. [HUD Element] 패턴 감지 알림 컴포넌트 (Pattern Detection Alert)
*   **목표:** AI의 개입(Intervention)이 일어나는 순간, 사용자에게 경고나 제안임을 인지시킵니다.
*   **레이아웃 컨셉:** 화면 좌측 상단 또는 우측 하단의 비활성 영역에 위치하는 투명한 오버레이 박스 형태.

| 상태 | 시각적 목표 및 의미 | 디자인 요소 정의 (Style Guide) |
| :--- | :--- | :--- |
| **[⚠️ 반복 조합 경고]** | 사용자의 패턴이 너무 단조롭다는 경고(Warning). AI가 개입하여 다양성을 강제적으로 유도. | - **배경:** 반투명 다크 그레이 (`#1A2438`, Opacity 0.7) <br>- **아이콘:** 삼각형 형태의 느낌표 (⚠️)와 함께, 그래프 패턴이 미세하게 흔들리는 애니메이션 적용.<br>- **색상 강조:** 경고성을 나타내는 '앰버 오렌지' (`#FFC107`).<br>- **텍스트 예시:** `[AI ALERT] Repetitive pattern detected. Consider introducing variety.` |
| **[✨ 스타일 제안/발견]** | 새로운 조합이나 잠재적인 가치를 발견했음을 알리는 긍정적 개입(Suggestion). | - **배경:** 반투명 다크 그레이 (`#1A2438`, Opacity 0.7) <br>- **아이콘:** 빛이 퍼져나가는 형태 (✨) 또는 연결된 두 개의 점과 선.<br>- **색상 강조:** 발견/희망을 나타내는 '시안 블루' (`#00BCD4`).<br>- **텍스트 예시:** `[AI SUGGEST] Found a potential connection. Try pairing A with C.` |

#### 2. [HUD Element] 인터랙션 가이드라인
*   **Typography:** Inter, Regular (Weight: 300~500)
*   **Color Palette:**
    *   Primary BG: `#1A2438` (Deep Navy/Black)
    *   Accent Warning: `#FFC107` (Amber)
    *   Accent Suggestion: `#00BCD4` (Cyan Blue)
    *   Text Color: `#E0E0E0` (Off-White)

### 📑 Task 2: Smart Care Alert Modal 및 아이콘 세트 제작

OCR에서 추출된 라벨링 정보를 바탕으로, 사용자에게 과학적이고 필수적인 '관리 코칭'을 제공하는 모달입니다. 이 모달은 단순히 정보를 보여주는 것이 아니라, **상황 예측(Contextualization)**의 역할을 수행합니다.

#### 1. [Wireframe] Smart Care Alert Modal (모바일 기준)
*   **Trigger:** 사용자가 옷장 내 특정 아이템(예: 울 스카프)을 선택하거나, 코디를 완성했을 때.
*   **레이아웃 구조:**
    1.  **(헤더)** 앨범/아이템 이미지 + [닫기 버튼] (X).
    2.  **(Alert Title)** 가장 중요한 경고 메시지를 상단에 배치합니다. (예: **"🚨 관리 필수 알림: 습도 변화 예측"**)
    3.  **(Core Detail)** OCR에서 추출된 원본 정보와 현재 상황을 대조하여 문제점을 명확히 합니다.
        *   *Original Label:* "소재: 울(Wool) / 세탁: 드라이클리닝 권장"
        *   *Alert Logic:* (현재 날씨/습도 데이터 연동) $\rightarrow$ **[위험 요소]** 현재 습도(78%)는 울 소재 변형의 주요 원인입니다.
    4.  **(Actionable Tip)** 가장 중요한 해결책을 제시합니다. (예: "→ *권장 보관법:* 통기성이 좋은 제습제와 함께, 옷걸이에 걸어 건조한 곳에 보관하세요.")
    5.  **(Icon/Severity Bar):** 경고 레벨을 시각적으로 표시하는 바 또는 아이콘 그룹.

#### 2. [Icon Set] Smart Care Alert 아이콘 및 심볼 정의
모든 아이콘은 **선형(Line Art)** 스타일을 유지하여 미니멀하고 기술적인 느낌을 강화합니다. (Stroke Weight: 1.5px)

| 알림 유형 | 아이콘/심볼 | 설명 및 사용 컨텍스트 | 색상 코드 |
| :--- | :--- | :--- | :--- |
| **🚨 위험 경고** | `[!]` 또는 세모난 삼각형에 느낌표 (`!`) | 즉각적인 조치가 필요한 경우 (예: 습도, 오염). | `#E53935` (Alert Red) |
| **⚠️ 주의 알림** | `[`와 `]` 사이에 점이 있는 원형 경고 (`…`) | 관리 시 유의해야 할 상황 (예: 계절 변화, 소재 민감성). | `#FFC107` (Warning Amber) |
| **✅ 권장 사항** | 체크 표시가 되어 있는 잎사귀 또는 방패 (`🛡️`) | 최적의 보관 및 사용 방법 제시. | `#4CAF50` (Success Green) |
| **🌡️ 환경 분석** | 온도계와 습도 게이지 결합 형태 | 현재 외부 환경 요인(날씨, 기온)을 기준으로 경고를 띄울 때. | `#2196F3` (Info Blue) |
