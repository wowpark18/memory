## ✨ EPHA 브랜드 비주얼 컨셉 보드 (Mood Board)
### **컨셉명:** The Unveiling Flow (개방되는 흐름)
*   **핵심 지향점:** '근원적인 막힘(Blockage)'을 겪는 사용자에게, 기술적으로 가장 간결하고 우아한 방식으로 **'통찰력과 해방감(Opening/Flow)'**을 제공하는 비주얼 언어.
*   **키워드 조합:** Minimalism (간결함) + Deep Tech (원리적 깊이) + Epiphany Light (깨달음의 빛).

---

### 🎨 1. 컬러 팔레트 (Color Palette) - '빛'과 '깊이'의 균형
*   **목표:** 과도하게 밝거나 어둡지 않은, 지적인 중립성을 기반으로 하되, 핵심 액션이나 인사이트가 발견되는 순간을 빛으로 표현합니다.

| 역할 | 색상명 (Conceptual Name) | HEX 코드 | 사용 원칙 및 의미 |
| :--- | :--- | :--- | :--- |
| **Primary Background** | Pure Canvas (순수 캔버스) | `#FFFFFF` | 모든 배경의 기본입니다. 절대적인 '여백'과 '간결함'을 상징하며, 시각적 피로도를 낮춥니다. |
| **Secondary Base** | Deep Ink (깊은 먹) | `#1A2038` | 텍스트, 주요 UI 요소, 배경의 깊이를 표현합니다. 순수한 검정(`#000000`)보다 부드러우면서도 무게감이 있어 '지성'을 대변합니다. |
| **Accent Color (The Open)** | Aura Blue (아우라 블루) | `#4A90E2` | 핵심 액션 버튼(CTA), 성공/완료 메시지, 데이터 흐름의 시작점을 표시합니다. 막혔던 것이 '열리는' 순간의 청명한 느낌을 상징합니다. (에바다 사명 직결) |
| **Highlight Color (Insight)** | Soft Glow (소프트 글로우) | `#B3E5FC` | 배경 위에 겹치는 빛, 그래프의 최고점, 새로운 인사이트가 발견되는 영역을 강조하는 '은은한 하이라이트'로 사용합니다. 절대 주색으로 쓰지 않습니다. |

### ✒️ 2. 타이포그래피 방향성 (Typography Direction) - '구조화된 명료함'
*   **목표:** 복잡한 내용을 다루지만, 마치 애플의 제품처럼 모든 문자가 직관적이고 읽기 쉬워야 합니다. 군더더기 없는 기하학적 산세리프(Geometric Sans-serif) 계열을 채택합니다.

| 역할 | 추천 폰트 (Korean/Global) | 사용 원칙 및 활용법 |
| :--- | :--- | :--- |
| **Headlines / Impact** | Pretendard Bold 또는 Noto Sans KR Black | *활용:* 제품 이름, 핵심 가치 문구(슬로건), 큰 타이틀. <br>*원칙:* 대문자 사용을 최소화하고, 크기와 무게 변화를 통해 위계를 만듭니다. 폰트 자체의 장식성보다 **간격(Tracking) 조절**에 집중하여 여백으로 임팩트를 만듭니다. |
| **Body Text / Detail** | Pretendard Regular 또는 system default (San Francisco/Roboto 등) | *활용:* 설명 문구, 본문 내용, 데이터 값. <br>*원칙:* 가독성이 최우선입니다. 16px 이상의 크기를 유지하며, 단락 간 충분한 여백(Line Height: 1.5~1.6)을 확보합니다. |
| **코드/데이터** | JetBrains Mono 또는 Source Code Pro | *활용:* 기술적 원리 설명, 데이터 시각화의 축 레이블. <br>*원칙:* 일반 텍스트와 명확히 구분되는 '기계적인' 느낌으로 전문성을 강조합니다. |

### 🖼️ 3. 비주얼 모티프 및 스타일 (Visual Motif & Style) - '추상적 원리의 시각화'
*   **목표:** 사람이 아닌, **‘정보와 개념의 흐름’**을 보여주는 방식으로 디자인 합니다.

1.  **빛과 연결 (Light & Connection):**
    *   **연출 방식:** 빛이 사물을 비추는 것이 아니라, **개념들이 만나서 스스로 발광하는 듯한 연출**을 사용합니다.
    *   **요소:** 부드러운 그라디언트(Aura Blue → Soft Glow)를 사용하여 경계가 모호한 '흐름'이나 '연결고리'를 표현합니다.
2.  **데이터 구조화 (Geometric Abstraction):**
    *   **연출 방식:** 복잡한 정보를 단순한 **기하학적 도형(원, 삼각형, 선)**과 연결선으로 간소화하여 보여줍니다. 데이터 그 자체가 아닌, 데이터를 담는 '구조'를 디자인합니다.
3.  **미니멀리즘의 깊이 (Minimal Depth):**
    *   **연출 방식:** 평면적인 배경 위에 미세한 입자감(Grain)이나 뎁스 필드(Depth Field/Bokeh Effect)를 넣어, 시각적으로 '깊은 공간'에 있는 듯한 느낌을 부여합니다. 이는 단순함을 유지하면서도 기술적 깊이를 상징합니다.
