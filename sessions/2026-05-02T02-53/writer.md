## 🔗 마이크로 카피라이팅 세트: 정보 단절 해소 프로세스 (EPHA)

본 카피는 EPHA의 핵심 가치인 '정보 보유'가 아닌 **'지식 연결 과정'** 자체를 상품화하는 데 초점을 맞추었습니다. 모든 문구는 지적이고 권위적이면서도, 사용자에게 공감적인 '해방감'을 전달하도록 설계되었습니다.

---

### 🎯 A. 랜딩 페이지/웹사이트 메인 카피 (Headline Focus)

**목표:** 사용자가 현재 겪는 막연한 불편함(Pain)을 명확히 지적하고, EPHA가 그 해답임을 선언적으로 보여줍니다. 짧고 간결하게 'Before'와 'After'를 대비시킵니다.

| 카피 유형 | Pain Point (문제 정의) | Solution/Benefit (해결 경험) |
| :--- | :--- | :--- |
| **헤드라인 1** | "지식은 있는데, 연결고리가 보이지 않나요?" (The Fragmented Knowledge) | EPHA. 파편화된 지식을 통찰력의 경로로 재구축합니다. |
| **헤드라인 2** | "정보 과부하 시대, 무엇이 핵심인지 알 수 없어 막막한가요?" (The Overwhelm Gap) | 데이터에 압도되지 마세요. EPHA는 정보 흐름 자체를 시각화합니다. |
| **서브 카피** | 단순히 지식을 모으는 것을 넘어, 지식들이 서로 만나 진화하는 '과정'을 경험하세요. |

### 🎬 B. 유튜브 스크립트 오프닝 후킹 (Video Hook)

**목표:** 시청자의 주의를 즉각적으로 사로잡아(Hook), EPHA의 기능이 아닌 **'사용자의 지적 변화 과정'**에 집중하게 만듭니다. (권위적이되, 공감적이어야 함)

**(화면 연출 가이드: 빠른 컷 전환, 복잡한 그래프가 깨지고 단순한 경로로 정리되는 시각 효과)**

> **[스크립트 초안]**
> "당신은 지금 지식의 홍수 속에서 헤매고 있습니다. 중요한 정보는 분명히 가지고 있지만, 그 조각들이 어떻게 연결되어 하나의 거대한 '통찰'을 만들어내는지 아무도 알려주지 않았죠. 수많은 자료와 노트가 쌓일수록, 오히려 무엇이 핵심인지 감각적으로 잊어버립니다. **진정한 문제는 지식의 부족함이 아닙니다.** 그것은 바로, 정보들 사이를 흐르는 ‘관계성’과 그 연결 과정을 볼 수 없다는 겁니다."
>
> *(잠시 멈춤)*
>
> "EPHA는 단순한 저장소가 아닙니다. 당신의 모든 지식을 하나의 '지능형 프로세스 시스템'으로 재조립하여, **가장 빠르고 논리적인 해답의 경로**를 시각적으로 증명해냅니다."

### 🛠️ C. 인앱/도움말 안내 텍스트 (In-App Micro Copy)

**목표:** 사용자가 기능을 사용할 때마다 EPHA가 단순한 도구가 아닌, '사고방식 전환'을 돕는 시스템임을 상기시킵니다. 간결하고 전문적이어야 합니다.

| 활용 상황 | 카피라이팅 (Tone: 지능형, 프로세스 중심) |
| :--- | :--- |
| **[정보 입력 시]** | "파편화된 정보를 주입하세요. EPHA가 맥락(Context)을 찾아 연결의 가능성을 탐색합니다." |
| **[분석/처리 과정 중]** | "지식 간 관계성 분석 진행 중... 단순 검색 결과가 아닌, **논리적 흐름과 진화 경로**를 추적하고 있습니다." |
| **[결과 도출 시 (Aha Moment)]** | "💡 연결 고리를 발견했습니다. 이 정보들은 개별적인 사실이 아닙니다. 하나의 [통합된 프로세스]로 작용하며 새로운 인사이트를 창출합니다." |
