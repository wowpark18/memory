## 🚀 [Look-UP] 정보 단절 해소 프로세스 인터랙티브 웹 프로토타입 블루프린트 (Mockup)
본 산출물은 개발팀이 구축한 Mock Data 환경(`knowledgeGraphMock.ts`)을 기반으로, 사용자가 '정보 Gap'를 인지하고 EPHA의 연결 프로세스를 경험하며 최종적인 'Aha Moment'에 도달하는 전 과정을 시각화하는 고화질 와이어프레임/인터랙티브 프로토타입 스펙입니다. 모든 애니메이션과 상태 전환(State Transition)을 구체적으로 정의합니다.

---

### 🎯 목표: 정보 단절 → 과정 발견 → 통찰력 확립 (The Aha Moment) 시각적 증명
**핵심 원칙:** 추상적인 지식 전달이 아닌, **'정보가 연결되는 순간의 드라마틱한 경험'** 자체를 제품의 핵심 가치로 만듭니다.

| 영역 | Mock Data Source | 시각적 기능 (Visual Function) | 핵심 애니메이션 콘셉트 |
| :--- | :--- | :--- | :--- |
| **메인 그래프뷰** | `MOCK_NODES`, `MOCK_EDGES` | 전체 지식 생태계의 지도. 현재 Gap을 시각적으로 강조. | 미세한 그리드 오버레이와 배경 노이즈 필터 효과 (Deep Tech 분위기 유지) |

### 🔄 인터랙티브 플로우 및 상태별 상세 스펙 (User Journey)

#### 💡 State 1: Pain Point 감지 (The Gap - 초기 화면)
*   **상황:** 사용자가 특정 주제에 대해 검색하지만, 핵심 정보를 찾기 위해 여러 출처를 떠돌고 있음.
*   **화면 구성:**
    *   배경은 미니멀한 다크 모드(Deep Navy: `#0A192F`)로 설정하고, 중앙에 불규칙하게 배열된 노이즈성 데이터 파편들(미완성 노드)을 배치합니다.
    *   가장자리에는 명확한 경계 없이 어둡고 불안정한 'Gap Zone'이 시각적으로 존재함을 강조합니다.
    *   **[UI 컴포넌트] Gap Alert Modal:** 화면 중앙에 "정보의 단절로 인해 중요한 연결 고리가 누락되었습니다."와 같은 문구가 옅은 에러 메시지 형태로 깜빡이며 표시됩니다.
*   **애니메이션:** 배경 전체에 낮은 주파수의 그리드 패턴이 깔려있으나, 이 그리드가 Gap Zone에서 주기적으로 '깨지는' 듯한(Disruption) 시각 효과를 보여줍니다.

#### ✨ State 2: EPHA의 개입 및 과정 추적 (The Discovery - 핵심 상호작용)
*   **상황:** 사용자가 특정 노드 A를 클릭하여 프로세스 분석을 요청합니다.
*   **액션:** **사용자 > [노드 A] 클릭 → [Edge 연결 시작]**
*   **시각 변화:**
    1.  **활성화:** 노드 A가 전경으로 튀어나오듯(Pop-up) 부각되며, 주변의 다른 노드들이 이 노드를 향해 미세하게 끌어당겨지는(Gravitational Pull) 애니메이션이 발생합니다.
    2.  **프로세스 시작 (Gap to Edge):** Gap Zone 내부에서 연결을 시도하는 **'빛나는 에지(Luminous Edge)'**가 발사됩니다. 이 빛은 `#66FFFF` 계열의 시안 그라데이션으로, 마치 전기 신호처럼 노이즈를 뚫고 전진합니다.
    3.  **엣지 라벨링:** 에지가 이동하는 경로(Edge) 위에 `processType`에 따른 설명을 실시간으로 오버레이 합니다. (예: *[CAUSAL] → 인과적 추론 과정* / *[SUPPLEMENTARY] → 보충 지식 연결*)
    4.  **중간 노드 발견:** 에지 경로 중간에 새로운 '연결점(Intermediate Node)'이 발견되면, 이 노드가 빛을 받아 점차 선명해지는 애니메이션(`Fade In` + `Bloom Effect`)을 거칩니다.

#### 🌟 State 3: 정보 단절 해소 및 통찰 (Aha Moment - 최종 결과)
*   **상황:** 에지가 목표 노드 B에 성공적으로 연결됩니다. Gap Zone이 완전히 메워지며 지식이 구조화되는 순간입니다.
*   **액션:** **Edge 도착 → [노드 B] 연결 완료.**
*   **시각 변화 (가장 중요):**
    1.  **완성 애니메이션:** 빛나는 에지가 노드 B에 도달하는 순간, `#FFD700` 계열의 골드 하이라이트가 노드 B를 감싸며(Halo Effect) 연결을 확정합니다. 이와 동시에 배경의 Gap Zone 전체가 부드럽게 사라지거나, 명확하고 질서 있는 그리드로 재구성됩니다.
    2.  **결과물 제시:** 화면 하단에 **[Key Takeaways]** 섹션이 깔끔하게 펼쳐지며, 노드 A → Edge → 노드 B의 논리적 경로를 요약한 구조 다이어그램(Flow Diagram)을 보여줍니다. 이 과정은 일반적인 텍스트 나열 방식보다 훨씬 강력한 시각적 만족감을 제공해야 합니다.
*   **사운드 디자인 가이드:** State 3 도달 시, '쉬이익-'하는 에지 사운드가 부드럽고 명료한 '띵' 하는 성공음(Ding)으로 마무리되도록 설계합니다.
