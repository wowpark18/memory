# 📐 EPHA 랜딩 페이지 - 인터랙티브 와이어프레임 및 디자인 시스템 가이드
(The $L_{Total}$ Crisis Funnel)

## 1. [핵심] 비주얼 디자인 시스템 정의 (Design System Definition)

| 요소 | 이름/코드 | 사용 목적 | 심리적 효과 | 적용 예시 |
| :--- | :--- | :--- | :--- | :--- |
| **Primary Color** | `#0A1931` (Deep Navy) | 배경, 텍스트 기본색. 전문성 및 신뢰도 확보. | 안정감, 깊이, 권위 (Depth, Authority). | 히어로 섹션 배경, 헤더 바. |
| **Secondary Color** | `#4DCCFF` (Cyan Blue) | 연결선(Edge), 성공적인 발견 지점, 핵심 정보 강조. | 통찰력, 해결책 제시, 깨달음 (Aha Moment). | 그래프의 정상 경로, CTA 버튼. |
| **Warning Color** | `#FFA500` (Amber Orange) | 잠재적 리스크 경고, 주의 단계 ($L_{Info}$ 발생). | 긴장감 고조, '뭔가 잘못됐다'는 자각 유도. | 데이터 변수 그래프의 오버레이. |
| **Critical Color** | `#E74C3C` (Bright Red) | 위기 임계점 도달, 재무적 손실 발생 ($L_{Total}$ 폭발). | 공포, 즉각적인 위험(Imminent Danger), 경고. | Paywall 영역의 배경/그래프 스파이크. |
| **Typography** | Pretendard (또는 Inter) | 가독성이 높은 산세리프 계열. 데이터 시각화에 최적화. | 현대적, 간결함, 기술적 정확성. | 모든 텍스트 요소. |

## 2. [와이어프레임 단계별 플로우] User Journey & Interaction Blueprint

### **Phase 1: Hook (안일한 시작) - The Familiar World**
*   **목표:** 사용자의 현재 상황에 공감하며 진입 장벽을 낮춘다.
*   **시각적 연출:** 밝고 깨끗한 배경 (White/Light Gray). 평온하고 안정적인 폰트 색상 유지.
*   **핵심 요소:** '당신은 충분히 준비했다고 생각하셨나요?'라는 질문과 함께, 사용자가 직접 자신의 자산군(예: 주식, 채권)을 체크하는 간단한 인터랙션 위젯 배치. (실제로는 단순 데이터 입력으로 시작).

### **Phase 2: Problem Recognition (불안 증폭) - The Data Leak**
*   **목표:** 사용자가 스스로 '정보의 불완전성'을 인지하게 만든다.
*   **시각적 연출 변화:** 배경에 은은한 그리드 오버레이(Grid Overlay, `#0A1931` 투명도 10%)를 적용하여 기술적인 느낌 부여.
*   **핵심 인터랙션: $L_{Total}$ 시뮬레이터 도입.**
    1.  사용자가 자신의 포트폴리오 A를 입력하면, 시스템은 기준선(Baseline) 그래프를 보여준다.
    2.  스크롤을 내릴 때마다 **Researcher가 제공한 리스크 변수 (규제 변화, 금리 인상 등)**가 하나씩 활성화되면서 Baseline 위에 '추가 손실 곡선'이 덧씌워진다.
    3.  **UI 액션:** 각 리스크 변수가 나타날 때마다 관련 문구와 함께 `[L_{Risk}]` 값이 **Amber Orange**로 증가하는 그래프를 실시간으로 보여준다. (사용자 스스로 '추가 비용'을 체감하게 함).

### **Phase 3: Crisis Escalation & Paywall Zone (절박함 극대화)**
*   **목표:** 통제할 수 없는 재무적 공포(Controlled Panic)를 유발하고, 외부 전문가의 개입만이 출구임을 각인시킨다.
*   **시각적 연출 변화:** **급격한 색상 전환.** 배경이 Deep Navy(`#0A1931`)로 어두워지며, 주변 UI 요소에 `#E74C3C` (Critical Red)의 플리커(Flicker) 효과를 적용한다.
*   **핵심 인터랙션: The Tipping Point 경고.**
    1.  그래프가 특정 임계점($L_{Threshold}$)을 넘어서는 순간, **전체 화면에 빨간색 경고 오버레이 (Red Overlay)**가 팝업된다. (애니메이션 필수).
    2.  **경고 메시지:** "🚨 [시스템 오류]: 현재 $L_{Total}$ 값은 귀하의 내부 정보 처리 능력으로 감당할 수 있는 임계점을 초과했습니다."
    3.  이와 함께, **$L_{Regulatory}$ 및 $L_{Financial}$가 결합된 복합 손실 시나리오**가 애니메이션화되어 폭발적으로 증가하는 그래프를 보여준다. (빨간색 스파이크).

## 3. [필수 컴포넌트 디자인 상세] 위기 경고 모달 (The Paywall Modal)

이 컴포넌트는 단순한 결제창이 아니라, **'해결책 부재로 인한 공포 자극 장치'**입니다.

*   **모달 제목:** "🛑 정보 단절 위험 경고: 전문가 개입이 필수적입니다."
*   **비주얼 특징:** 배경에 Red Overlay가 깔려 있어 심리적 압박을 유지합니다. 모달 테두리 주변에 깜빡이는(Pulsing) 효과를 적용하여 긴급함을 강조합니다.
*   **내용 구성 (3단계):**
    1.  **문제 재확인:** "현재 상황은 단순한 '지식 부족'이 아닙니다. 구조적 시스템 결함으로 인한 **통제 불능의 공포($L_{Total}$) 상태입니다.**" (빨간 글씨 강조).
    2.  **솔루션 제시:** "이 위기 임계점을 돌파하고 손실을 회복하기 위해서는, 복합적인 변수를 분석할 수 있는 **외부 전문가의 맞춤형 개입(EPHA 컨설팅)**만이 유일한 해결책입니다." (Cyan Blue 강조).
    3.  **CTA 버튼:** "지금 즉시 $L_{Total}$ 진단 요청하기" (가장 눈에 띄는 Primary CTA, `#4DCCFF` 배경의 빨간 경계선).
