# ✨ EPHA 핵심 프로세스 시각화 컴포넌트 시스템 가이드

## 1. 컬러 및 타이포그래피 정의 (Palette & Typography)

| 요소 | 용도 | HEX Code | 설명 |
| :--- | :--- | :--- | :--- |
| **Primary Background** | 기본 배경, 신뢰감 확보 | `#0A192F` (딥 네이비/블랙) | 기술적 깊이와 안정감을 주는 어두운 배경색. |
| **Accent - Connection** | 연결성, 해결책, 통찰 (Aha Moment) | `#4DD0E1` (사이트 시안 블루 계열) | 빛나는 경로, 발견의 순간을 상징하는 주 강조색. 그라데이션 사용 필수. |

*   **본문/설명:** Pretendard Regular (높은 가독성 확보)

## 2. 컴포넌트별 상세 스펙 및 사용법

### A. [NODE] 정보 노드 (지식의 파편, Concept Unit)
*   **디자인 형태:** 사각형 또는 원형의 깔끔한 카드/버블 형태. 모서리를 살짝 둥글게 처리하여 딱딱함을 완화합니다.
*   **기본 스펙:** `Width: 200px`, `Height: 80px` (직사각형). 배경색은 라이트 그레이 (`#EEEEEE`)를 사용하되, 다크 모드에서는 반투명한 어두운 박스 처리하여 '정보 조각' 같은 느낌을 부여합니다.
*   **텍스트:** 노드 제목(Bold), 설명 캡션(Regular)으로 구성.

### B. [GAP] 정보 단절 영역 (Information Silo / Missing Link)
*   **디자인 스펙:** 노드와 노드 사이에 배치되는 공간적 결함 자체를 시각화합니다.
    *   **색상/효과:** `Danger - Gap` 색상을 배경에 얇게 오버레이 하거나, 연결선 자체가 **노이즈(Noise)와 블러 효과(Gaussian Blur)**로 가득 차서 흐릿하게 보이도록 처리합니다.
    *   **애니메이션 (디지털):** 데이터가 통과하려 할 때 '지지직'거리는 노이즈 사운드 및 시각적 글리치(Glitch) 애니메이션을 추가하여 정보의 손실 경험을 강조합니다.
    *   **시각적 메시지:** "Context Missing" 또는 "연결 경로 분석 필요"라는 붉은색 경고 문구를 배치합니다.

### C. [PATH] 연결 프로세스 (Connection Path / The Process Flow)
*   **디자인 스펙:** Gap을 뚫고 지나가거나, Gap을 건너뛰며 노드를 연결하는 '에지(Edge)' 또는 '흐름' 자체를 표현합니다.
    *   **색상/효과:** `Accent - Connection` 색상의 **밝은 사이언 그라데이션 (Gradient)**을 적용합니다. 단순히 선이 아니라, 마치 데이터가 흐르는 듯한 **미세한 광선(Light Streak) 효과**와 함께 부드러운 곡선형 트랜지션(`Ease-in-out`)을 사용합니다.
    *   **추가 요소:** 연결 경로 위로 작은 '데이터 패킷' 아이콘이 순차적으로 움직이는 애니메이션을 추가하여 역동성을 부여합니다.

## 3. 미디어별 적용 가이드라인 (Adaptation Guide)

| 플랫폼 | 주 컴포넌트 사용 방식 | 레이아웃/배치 전략 | 핵심 전달 감성 |
| :--- | :--- | :--- | :--- |
| **블로그 이미지** | 정지된 다이어그램 형태. (Knowledge Flow Diagram) | 3~5개의 노드를 배치하고, Gap과 Path를 명확히 구분하는 *좌우/상하* 플로우 구조 사용. 배경은 미니멀한 그리드 오버레이(`Opacity: 10%`) 유지. | 학술적 근거 제시, 신뢰성 확보. |
| **유튜브 배경 그래픽** | 애니메이션 기반의 다이나믹 시퀀스. (Motion Graphics) | 노드가 갑자기 나타나고(Fade In), Gap에서 Path가 폭발적으로 생성되며 연결되는 과정 전체를 15~30초에 걸쳐 연출. 영상 전반에 사이언 컬러가 주도적임. | 문제의 극복, 지능적인 해결책 제시. |
| **인스타그램 카드뉴스** | 슬라이드별 스토리텔링 컴포넌트. (Sequential Storytelling) | 1장: Pain(Gap만 클로즈업). 2장: 이론(노드와 Gap 병치). 3장: Solution(Path가 노드를 연결하며 빛남). 간결한 아이콘과 명료한 텍스트 위주 구성. | 공감 유발, 빠르고 직관적인 깨달음 (Aha Moment). |
